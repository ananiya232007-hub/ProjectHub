# ProjectHub - Project Management & Task Tracking System

## 🚀 Overview
ProjectHub is a full-stack enterprise-grade project management application designed to help teams efficiently manage projects, tasks, and team collaboration. Built with modern web technologies, it provides a professional dashboard for tracking project progress, managing tasks, and monitoring team performance.

## ✨ Features

### 🔐 Authentication
- Secure JWT-based authentication
- User registration with role selection (Project Manager, Developer, Designer, Tester)
- Login with email/password
- Protected routes and session management

### 📊 Dashboard
- Real-time project statistics
- Welcome section with personalized user greeting
- Professional UI with light/dark theme support
- Responsive design for all devices

### 📁 Project Management
- Create, Read, Update, Delete projects
- Project status tracking (Active, In Progress, Completed)
- Project descriptions and details
- Recent projects overview

### 👤 User Profile
- Profile card with user details
- Avatar with user's first letter
- Role-based access
- Member since tracking

### 🎨 UI/UX Features
- Glassmorphism design
- Animated particles background
- Dark/Light mode toggle
- Responsive layout
- Font Awesome icons
- Smooth animations and transitions

## 🛠️ Tech Stack

### Frontend
- HTML5
- CSS3 (Custom with CSS variables)
- JavaScript (Vanilla)
- Font Awesome Icons
- Google Fonts (Inter)

### Backend
- Node.js
- Express.js
- MongoDB with Mongoose
- JWT for authentication
- bcrypt for password hashing
